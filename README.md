vissort.vim
===========

vissort.vim for vundle install

see doc folder for license and author info

update 3/2025
-----------
I still use this in my neovim setup.

Lazy config:
========

```lua
{
  'navicore/vissort.vim',
}
```

I suggest using `vis` with `vissort`:

```lua
{
  'navicore/vis.vim',
  'navicore/vissort.vim',
}
```
